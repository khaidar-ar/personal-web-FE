About Me

Saya akrab dipanggil "Khaidar", seorang mahasiswa teknik informatika. Saya mulai mempelajari bidang informatika dengan mempelajari bahasa pemrograman java dan spring-boot untuk pembelajaran tingkat lanjut. Disisi pengembangan web, saya mempelajari HTML5, CSS3 dan vanilla Javascript. Kedepan saya rasa akan lebih condong ke arah back-end karena sedari awal saya kurang begitu memiliki taste yang berhubungan dengan desain antarmuka pengguna.
